# 🎬 Netflix-Style Executive Dashboard 

> Executive streaming analytics dashboard focused on subscriber growth, churn prediction, engagement trends, retention KPIs, and executive-level business intelligence.

---

# 📊 Executive Overview

This project simulates how streaming companies like Netflix monitor:

- subscriber growth
- churn risk
- engagement behavior
- retention performance
- watch-time analytics

The dashboard combines Python analytics, KPI reporting, and executive-style visualizations to replicate real-world streaming analytics workflows.

# 🚀 Features

✅ Synthetic streaming subscriber dataset generation  
✅ Churn prediction analytics  
✅ Executive KPI dashboards  
✅ Retention trend analysis  
✅ Watch-time engagement metrics  
✅ Device usage insights  
✅ Netflix-style visuals  
✅ Streamlit dashboard integration
